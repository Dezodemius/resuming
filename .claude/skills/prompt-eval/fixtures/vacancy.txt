Backend-разработчик (Python) — финтех-платформа «ПэйФлоу»

О компании: «ПэйФлоу» — растущий финтех-сервис приёма онлайн-платежей для среднего бизнеса. Обрабатываем более 2 млн транзакций в месяц, команда разработки — 25 человек.

Чем предстоит заниматься:
- Разработка и поддержка платёжного ядра: приём платежей, маршрутизация по эквайерам, сверка и возвраты;
- Проектирование внутренних REST API для личного кабинета мерчантов и антифрод-сервиса;
- Оптимизация работы с базой данных при росте нагрузки (PostgreSQL, до 1,5 тыс. RPS в пике);
- Написание интеграций с банковскими API и платёжными провайдерами (ЮKassa, СБП);
- Участие в код-ревью и развитии инженерных практик команды.

Требования:
- Коммерческий опыт разработки на Python от 3 лет;
- Уверенное знание FastAPI или Django, понимание asyncio;
- Опыт работы с PostgreSQL: проектирование схем, оптимизация запросов, индексы;
- Опыт работы с брокерами сообщений (RabbitMQ или Kafka) и Redis;
- Привычка покрывать код тестами (pytest), опыт работы с CI/CD;
- Docker как ежедневный инструмент.

Будет плюсом:
- Опыт интеграции платёжных шлюзов и обработки вебхуков;
- Опыт работы в финтехе или с высоконагруженными системами;
- Базовое знание Kubernetes;
- Английский на уровне чтения технической документации.

Условия: гибрид (офис в Москве, 2 дня в неделю), ДМС, вилка 280–350 тыс. руб.
